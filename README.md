# File Size Calculator

This is a small tool I made to convert file sizes and check how big selected files are.

## What it can do

- Convert KB, MB, GB and TB
- Show total size of selected files
- Give simple and clear output

-----> How to use

For converting:
- enter a number
- choose the unit
- click convert

For checking files:
- select files
- click check files
- it shows total size

------> Why I made this

I wanted to build something simple but useful.  
I often needed to convert file sizes while working, so I thought of making a small tool for it instead of searching every time.

Also used this as practice to improve basic JS and UI.


-----> Progress
