File size reference

KB

Kilobyte is used for small files like text files.

MB

Megabyte is used for photos and documents.

GB

Gigabyte is used for videos and applications.

TB

Terabyte is used for large storage devices.

Basic conversion rules

  • 1 MB = 1024 KB
  • 1 GB = 1024 MB
  • 1 TB = 1024 GB
